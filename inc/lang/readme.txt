                       Language files
			         	(files needing translation)
                       ==============
All the files named below need translation from fr or en files.
If they are found  in a langage pack older than 4.5 release they need to be updated
according fr or en files of this pack:

- under inc/lang
  --------------
  - xx-admin.inc : all the strings used with admin scripts
  - xx-help.htm  : the on-line help file for editors
    MAIND, don't change number of lines or hight !!!
  - xx-special.js : the script to find catholic feasts and public holidays
    You can add your own feats or public holidays in $ferie list
    MAIND, if you add or substrac items in $ferie take care of ponctuation 
  - xx-web.inc    : the strings used with all other scripts	
  
- under inc/rtef/lang
  -------------------
  - xx.js : language strings for the postguest WYSIWYG editor (rtef).
	
- under /inc/fckeditor/editor/lang
  --------------------------------
  - xx.js : language strings for the admin WYSIWYG editor (Fckeditor). 
    Take care from mistakes between minus and MAJOR letters (and ponctuation of course!).
    
If there is no file according your language and you don't know how to translate 
you must copy fr or en file to your own language (for example: en-admin.inc ==> de-admin.inc,
en-lang.js ==> de-lang.js, en.js ==> de.js and so on).
